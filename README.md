# Take-Home Test for Chainalysis

## Stack:

- Backend: Python + Flask
- Frontend: React

## Exchanges:

1. Kraken
2. Gemini

## Build:

`cd chain-api && python3 -m venv .env && source .env/bin/activate && pip install -r requirements.txt && deactivate && cd ..`

## Run:

`npm install`
`yarn start`
`yarn start-api`
